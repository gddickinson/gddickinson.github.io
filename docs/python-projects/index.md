## Scientific & Microscopy Tools

### FLIKA Plugins
Professional plugins for fluorescence microscopy analysis.

#### [thunderSTORM](flika-plugins/thunderstorm.md)
Single Molecule Localization Microscopy (SMLM) analysis implementing thunderSTORM algorithms within FLIKA.

**Features**: Localization of single molecules • Super-resolution reconstruction • Drift correction • Batch processing

#### [SPT Batch Analysis](flika-plugins/spt-batch.md)
Automated single particle tracking with batch processing capabilities.

**Features**: Multiple tracking algorithms • U-Track integration • Gap closing • Interpolation methods

#### [Cell Edge Movement](flika-plugins/cell-edge.md)
Analysis of cell membrane dynamics and protein localization correlation.

**Features**: Edge detection • Velocity calculations • PIEZO1 protein analysis • Batch processing

#### [Mask Editor](flika-plugins/mask-editor.md)
Interactive ROI and mask editing tool for FLIKA.

**Features**: Frame-by-frame editing • Boolean operations • Multiple selection tools • Morphological operations
