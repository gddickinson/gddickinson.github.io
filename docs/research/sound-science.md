## Projects

### Lower Colorado River Multi-Species Conservation Program

Development and updates to ecological models for California Black Rail conservation in the Lower Colorado River region.

- Supervised machine learning for vegetation classification

- Temporal analysis of habitat changes
- Geospatial data integration

#### Methodology
- Classification of vegetation types from satellite imagery and LiDAR
- Temporal analysis of habitat availability
- Integration with species distribution data
- Automated processing pipelines
- Quality control and validation
